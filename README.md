# Ja eller Nei? 🤪

En liten morsom React-side med et spørsmål til Merethe. Trykk **Ja** så feirer vi med et regn av hjerter 💖 — men prøver du å trykke **Nei**, stikker knappen unna hver eneste gang. Umulig å si nei! 🙅

## Teknologi
- ⚛️ React 19
- ⚡ Vite
- 🎨 Glassmorphism-design med animert gradient og hjerte-konfetti

## Kjør lokalt
```bash
npm install
npm run dev
```

## Bygg for produksjon
```bash
npm run build
```

Siden publiseres automatisk til GitHub Pages via GitHub Actions ved push til `main`.
Live: https://rervik.github.io/ja-eller-nei/
